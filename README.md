Welcome to the project's GitHub repository. This repository serves as the core project's central location for issue tracking and feature development.

If you're not a developer or contributor, please use [LifterLMS plugin page][link-wp-repo] at WordPress.org.


### Getting Help and Support

GitHub is for bug reports and contributions only! If you have a support question or a request for a customization this is not the right place to post it. Please refer to [LifterLMS Support][link-support] or the [community forums][link-support-forums]. If you're looking for help customizing LifterLMS, please consider hiring a [LifterLMS Expert][link-experts].


### Reporting a Security Vulnerability

Security issues and vulnerabilities should be responsibly disclosed directly to the LifterLMS core developers via email. Please see our [Security Policy](https://lifterlms.com/security) for details on disclosing a security vulnerability.


### Contributing

Contributions are welcome! For full contribution guidelines, please refer to the LifterLMS Core's [Contributor's guidelines][link-contributing].
